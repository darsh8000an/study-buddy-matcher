## 🚀 Quick Start Commands

### Run Backend:
```bash
cd backend
npm install
cp .env.example .env
# Edit .env with your MongoDB URI
npm run dev
```

### Run Frontend:
```bash
cd frontend
npm install
npm run dev
```

### Run Tests:
```bash
# Backend tests
cd backend
npm test

# E2E tests (ensure backend + frontend running)
cd frontend
```

